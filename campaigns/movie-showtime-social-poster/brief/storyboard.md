# Movie Showtime Social Poster Layout

This is a static `1080x1350` social poster rather than a timed video.

| Zone | Locked content |
| --- | --- |
| Header | Intact official logo only |
| Primary message | Category, gold `showtimes`, and region |
| Secondary message | Category-only support copy or optional selected movie title |
| Action | Fixed `Find showtimes` CTA |
| Footer | `dein-dein.com` and visible approved disclaimer |

The Dark Focus layout, spacing, colors, and content zones remain fixed across
every generation.
