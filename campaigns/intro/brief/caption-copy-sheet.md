# Caption And On-Screen Copy Sheet

| Scene | Approved Copy | Purpose |
| --- | --- | --- |
| Hook | `You. Your movies.` | Personal movie-discovery promise |
| Discovery | `Discover movies near you.` | Explain the first useful action |
| Language breadth | `Many languages. One place.` | Communicate supported breadth without an absolute claim |
| Showtime proof | `Find when and where it is showing.` | Explain the practical result |
| Provider handoff | `Continue to the official ticket provider.` | Explain the external handoff |
| Provider disclosure | `Tickets are sold by the official provider.` | Avoid implying Dein-Dein sells tickets |
| Closing CTA | `Find showtimes` | One approved next step |
| Closing URL | `dein-dein.com` | Official destination |

Prohibited copy: `Book Now`, `Buy tickets`, guaranteed availability, pricing, popularity, or ticket-sales claims.
